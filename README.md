# Python Flask Hello World for Vercel

This is a minimal Flask web app that displays 'Hello, World!' and is ready to deploy on Vercel.

## 本地运行

1. 安装依赖：
   ```
   pip install -r requirements.txt
   ```
2. 启动服务：
   ```
   python app.py
   ```
3. 访问 http://localhost:5000

## 部署到 Vercel

1. 安装 Vercel CLI（如未安装）：
   ```
   npm i -g vercel
   ```
2. 在项目目录下运行：
   ```
   vercel
   ```
3. 按提示完成部署。
